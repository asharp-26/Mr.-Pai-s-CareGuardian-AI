# Mr. Pai's CareGuardian AI

## Project Introduction

Mr. Pai's CareGuardian AI is a multi-agent elderly care assistant developed using Google Antigravity, Google ADK, and Gemini. The system supports medication scheduling, medication compliance tracking, emergency guidance, and healthcare assistance through specialized AI agents.
